# ピンキースレッジハンマー 🔨
**Pinky Sledgehammer — 固定観念を、にっこりブチ壊す**

---

## 概要

モヤモヤ・不満・愚痴を入力すると、4段階の解決策を提示するWebアプリ。
「問題を解決しに行かない。問題の発生条件ごと消すか、エネルギー源に変える」という思想が設計の核心。

**URL（予定）**: https://pinky-sledgehammer.vercel.app  
**開発開始**: 2026年3月19日 朝3時  
**開発者**: Masaki  

---

## 開発の経緯

### 発端
ChatGPTとのポケモン「カヌチャン（Tinkatink）」の会話がきっかけ。
カヌチャンの特徴「金属を集めてハンマーを作る職人気質な鍛冶屋」から、
「モヤモヤを叩いて形にする」というコンセプトが生まれた。

同日夜のセナリ学院授業で「WHO / WHAT / CORE VALUE」フォームを使った演習があり、
そこで感じた違和感が開発の火種になった。

> 「動画を見てフォームを埋める（読解の練習）じゃなくて、  
> フォームを埋めたら動画が生まれる（道具）にしたい」

授業でやっていたことの**逆向き**をアプリにする、という着想。

### コンセプトの進化
1. モヤモヤ錬金術師（真面目ver）→ WHO/WHAT/CORE VALUE の3枠出力
2. ピンキースレッジハンマー → カヌチャンのピンク×パープルに寄せたポップ版
3. 4段階化（NORMAL/JOY/EXTREME/HYPER）
4. 賢者降臨モード（人間関係・感情系に対応）

### 命名の由来
- **ピンキー**: カヌチャンのボディカラー（ホットピンク）
- **スレッジハンマー**: Peter Gabrielの楽曲「Sledgehammer」から連想
- **ラボは外した**: 「スレッジハンマーラボ」→「ピンキースレッジハンマー」に絞った

## 機能仕様

### 基本フロー
```
[モヤモヤ入力（テキスト）]
        ↓
   ⚒ ガチャン！
        ↓
[ESSENCE バナー] ← モヤモヤの本質を一言で
        ↓
[NORMAL カード]  ← 定番の解決策
[JOY カード]     ← ちょっとズレた視点
[EXTREME カード] ← スケボー的発想（世の中にあるが選ばれない方法）
        ↓
「まだモヤモヤしてる？」
        ↓
🔨 とどめの一撃、召し上がれ
        ↓
[HYPER EXTREME または 賢者降臨 カード]
```

### 4段階の定義

| レベル | 定義 | 足の怪我の例 |
|--------|------|-------------|
| NORMAL | 定番の解決策 | 松葉杖 |
| JOY | ちょっとズレた、でも納得感ある | おしゃれな義足・装具 |
| EXTREME | 世の中にあるが普通は選ばない | 電動車椅子でアクティブに |
| HYPER | 制約ごとひっくり返す、文脈の引越し | スケボーに乗る。上半身を鍛えて怪我前より速くなる |

### HYPER EXTREMEの本質
> 「問題を解決しに行かない。問題の発生条件ごと消すか、エネルギー源に変える。」
> 「改善・活用ではなく、文脈の引越し。」

### 賢者降臨モード
以下の条件でAIが自動判断して切り替わる：
- 人間関係・感情・承認欲求・孤独が核心にある
- テクノロジーや手法では本質的に届かない
- 「正しい答え」より「問いそのものを変える」が必要

賢者は名前を名乗らない。「ちょっと別の人に来てもらいました。」から始まる。

## HYPERの事例集（プロンプトに組み込み済み）

| モヤモヤ | HYPER |
|--------|-------|
| 事務処理が多くて大変 | 「その業務、そもそも必要ですか？」と問いかけて丸ごと廃止させる |
| マラソンの記録が伸びない | トレイルランに転向。「タイム」という物差しごと捨てる |
| 親に旅行の報告をしろと言われる | YouTuberになって1万人に先に届ける |
| 忘れ物が多くて怒られる | スマホと鍵だけで通勤する設計にする。忘れるものを消滅させる |
| ジムニーに乗れなくてもったいない | ジムニーに住む。距離ごと消せばフラストレーションは発生しない |

## 技術仕様

### 構成
- シングルHTML（index.html）
- Vanilla JS + CSS
- Anthropic API（claude-sonnet-4-20250514）

### API構造
- **1回目**（ガチャン！）: NORMAL/JOY/EXTREME の3枚 + ESSENCE を生成
- **2回目**（とどめの一撃）: HYPER または 賢者降臨 を別APIコールで生成

## デザイン思想

### キャラクター
オリジナルSVGキャラ（著作権フリー）。  
むっとした顔のピンク丸鍛冶屋ちゃんがふわふわ浮き、隣でハンマーが揺れる。

カヌチャンの世界観（ピンク×パープル×鍛冶）を借りつつ、著作権侵害なしで設計。

### トーン
> 「にっこりしながら、あなたの中の固定観念をブチ壊す」

- 温かく、押しつけがましくない
- 「正解」ではなく「選択肢の束」を渡す
- 怒る人もいるかもしれないが、眠っていた才能が爆発する可能性がある

### 参照イメージ
- 進め！電波少年（無茶振りをにっこり受け入れる）
- 松村邦洋（妙に納得感のあるとんでもない発言）
- ドラえもんの秘密道具（実在するが普通は選ばないソリューション）
