🔨 ピンキースレッジハンマー / Pinky Sledgehammer
「にっこり笑って、固定観念をブチ壊す。見た目はスイート、中身はビター。」
<ここにメインのスクリーンショット：アプリ全体が写っているもの>

「あー、モヤモヤする！」
そんな行き場のない不満や愚痴を、巨大なピンクのハンマーで叩いて、見たこともない形へ錬金するWebアプリです。

優しい励ましや、ありきたりな正論はここにはないかも。
このアプリが提供するのは、**「問題の発生条件ごと消し去る」か、「絶望をエネルギー源に変換する」**という、斜め上からの「文脈の引越し」です。

💎 3つの「ガチャン！」体験（サンプル事例）
あなたの悩みに合わせて、ハンマーの振るい方は3変化します。

① 【仕組みで解決】移動制限を「投資」へ
「足首を捻挫。3週間、松葉杖生活で絶望……」

NORMAL: 読書や映画で時間を潰す。

JOY: 上半身を鍛えて「腕立ての新記録」に挑む。

HYPER: 「3週間限定の最強集中環境」と定義。 外に出られない不自由を逆手に取り、高収益スキルを習得する人生のターニングポイントへ。

<ここに1枚目のスクショ：足の怪我のnormal, joy, extremeのコメント>

② 【物差しを捨てる】損得勘定を「ゲーム」へ
「山登りの後に名店つけ麺。晩飯代が浮くか損するかで迷う」

HYPER: 得損の物差しをゴミ箱へ。 「山登り＋つけ麺＋晩飯抜き」を一つの『満足度最大化プログラム』というゲームとして扱う。カロリーや金額ではなく「記憶の密度」で勝負する。

<ここに2枚目のスクショ：つけ麺のHyper>

③ 【魂の解体】おじさんのノイズを「滑稽な標本」へ
「配信で、女の子の良い発言をおじさんが遮って辛い……」

🌀 賢者降臨: 「そのムカつきは、あなた自身の『聞いてもらいたい』の裏返しでは？」と、核心のナイフを抜く。

視点の転換: 支えてくれている人に石を投げる人々の「矛盾」を、高所から観察する標本に変えてしまう。

<ここに3枚目のスクショ：おじさん配信の賢者>

⚒ このアプリの「正体」
最初は、仕事の仕組みやツールを改善する「ぶっ飛んだアイデア出し」として使えるでしょう。でも、使い続けるうちに気づくはずです。

このアプリは、**あなたの心を映し出す「鏡」**であることに。

過激なことは言いません。AIの倫理規定は守られています。
けれど、あなたの固定観念やプライドに対しては、容赦なくスレッジハンマーを振り下ろします。

最後に行き着くのは、**「矛盾だらけの世界を、どう面白がって受け止めるか」**という哲学対話。
楽しみながら使っているうちに、いつの間にかあなたの「心の柔軟性」が鍛えられている……かもしれません。

🛠 使い方
APIキーをセット: console.anthropic.com で取得したキーを保存（ローカル保存のみ）。

モヤモヤを叩き込む: 今抱えている不満を正直に書く。

ガチャン！と壊す: 4段階の回答と、必要なら「とどめの一撃」を召し上がる。
